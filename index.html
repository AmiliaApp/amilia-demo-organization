Embed in your store using an IFRAME

You can embed your Amilia Store directly on your site as demonstrated on this page. This is achieved with an IFRAME. Instructions are on our Support site or directly in Amilia under Store, Store Embedding.

WordPress Integration

If your website is on Wordpress, checkout the Amilia Store plugin.

Deep linking

The WordPress short-code integration supports deep linking via the amilia_url query string parameter. For example, say someone lands on this page:

https://www.example.com/store?amilia_url=https%3A%2F%2Fwww.amilia.com%2Fstore%2Fen%2Fusa-amilia-group%2Fshop%2Factivities%2F1870221

The iframe will open up on the URL passed through the query string: https://www.amilia.com/store/en/usa-amilia-group/shop/activities/1870221

Note: When you construct the query string URL, make sure to encode it.
